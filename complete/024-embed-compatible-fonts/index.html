Steps to embed your own fonts:

  1. Get your own font or download one from League of Movable Types
  2. Upload the font and generate the stylesheet css codes from Font Squirel generator
  3. copy all the .eot, .ttf, .otf, .woff font files in your css folder
  4. Cut and paste the css codes in your style.css
  5. ensure that the css font names and the actual file names are exactly the same. you might want to rename them in a simpler manner.
