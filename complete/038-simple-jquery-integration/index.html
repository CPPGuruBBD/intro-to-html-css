  • Fittext
  • Github codes and instructions for Fittext

Steps

  1. Download the javascript *.js file you are looking for. E.g. Fittext
  2. (optional) Link the necessary javascript library file when required. E.g. jQuery hosted by Google
  3. link the downloaded *.js file with proper relative hyperlink in the head tags. Ensure the file is actually in your project folder!
  4. Add the necessary html markup in your html/css as required
  5. call the javascript right before the end of body tag. (sometimes it depends whether before or after)
  6. (optional) add options to your javascript when necessary
  7. always check with the browser devtools console panel to see if there are any errors
  8. For fittextjs: if it's working properly, the sentence should still be in one line with smaller fonts as you resize the browser
