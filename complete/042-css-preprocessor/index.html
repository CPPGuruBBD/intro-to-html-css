    CSS Preprocessors

    • Scout App to compile SASS to CSS
    • http://sass-lang.com/ and Compass library
    • Stylus - another css preprocessor
    • LESS - another css preprocessor
    • Scout App - SASS and Compass compiler to css

    Steps

    1. create a file style.scss which has exactly the same contents as style.css
    2. import compass
    3. declare variables
    4. make some mixins for reused css code blocks
    5. implement nesting
    6. using the scout app compile SCSS to CSS
    7. (alternative) if you are comfortable with *nix command line and rubygems, then gem install sass compass
